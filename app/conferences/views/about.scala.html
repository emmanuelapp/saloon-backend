À propos

Ce projet est né du fait qu'il n'existe aucun moyen simple de savoir quelles conférences vont avoir lieu.
D'où l'idée de créer un site dédié. Et comme il est pénible de se tenir régulièrement au courant des nouveautés, vous avez maintenant une newsletter et un compte twitter pour suivre les conférences tech en France :D

L'objectif est de faire un site participatif afin d'être le plus exhaustif et réactif possible.
N'hésitez donc pas à ajouter vos propres conférences et corriger celles qui pourraient avoir des erreurs.
De plus, si des fonctionnalités essentielles vous manquent n'hésitez pas à les suggérer ;)

Enfin, le service se veut ouvert et propose une API pour accéder à ses données. Le fonctionnement de l'API est très proche de celui du site, voici les différents endpoints :

  • Liste des conférences à venir : /api/v1/conferences
  • Recherche dans toutes les conférences : /api/v1/conferences?q=scala (similaire à la recherche avancée)
    • q : texte recherché dans l'ensemble des champs
    • after (format: @Config.Application.dateFormat) : conférences qui démarrent après la date indiquée
    • before (format: @Config.Application.dateFormat) : conférences qui démarrent avant la date indiquée
    • tags (format: tag1,tag2,tag3) : conférences avec les tags indiqués
    • cfp (valeur: on) : filtre les conférences ayant un appel à orateur d'ouvert
    • tickets (valeur: on) : filtre les conférences ayant la billetterie ouverte
    • videos (valeur: on) : filtre les conférences ayant publié les vidéos des présentations
  • Liste des appels à orateurs ouverts : /api/v1/conferences/cfps
    • q : texte recherché dans l'ensemble des champs
  • Liste des présentations : /api/v1/conferences/presentations
    • q : texte recherché dans l'ensemble des champs
  • Liste des orateurs : /api/v1/conferences/persons
    • q : texte recherché dans l'ensemble des champs

Le site a été codé en Scala et son code est publié en Open Source sur le github de SalooN.
