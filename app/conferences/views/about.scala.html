About

Ce projet est né du fait qu'il n'existe aucun moyen facile de savoir quelles conférences vont avoir lieu.
D'où l'idée de créer un site dédié à ça. Et comme c'est pénible de venir voir régulèrement les nouveautés, vous avez une newsletter et un compte twitter pour suivre l'activité des conférences tech en France :D

L'objectif est de faire un site participatif pour pouvoir etre le plus exhaustif possible.
N'hésitez donc pas à ajouter vos propres conférences et corriger celles qui pourraient avoir des erreurs.
De plus, si des fonctionnalités essentielles vous manquent n'hésitez pas à les suggérer ;)

Enfin, voulant etre les plus ouvert possible, les données sont accessibles via une API :

  • Liste complète : /api/conferences
  • Recherche : /api/conferences/search?q=scala (similaire à la recherche sur le site)
    • q : texte recherché dans l'ensemble des champs
    • after (format: dd/MM/yyyy) : conférences qui démarrent après la date indiquée
    • before (format: dd/MM/yyyy) : conférences qui démarrent avant la date indiquée
    • tags (format: tag1,tag2,tag3) : conférences avec les tags indiqués
    • cfp (valeur: on) : filtre les conférences ayant un appel à orateur d'ouvert
    • tickets (valeur: on) : filtre les conférences ayant la billetterie ouverte
    • videos (valeur: on) : filtre les conférences ayant publié les vidéos des présentations

Le code de ce site est publié en Open Source sur le github de SalooN.
